# 🍗 Chubby's Chicken — Content Intelligence Dashboard

Built by Hyperwave Studios

## What This Is

A content strategy dashboard for Chubby's Hot Chicken (Fontana, CA) that includes:

- **📊 Analytics** — Upload Instagram Insights CSV to track growth, engagement, top posts
- **🔥 Trending Intel** — Trending audios, formats, and hashtags in the Nashville hot chicken space
- **👀 Competitor Watch** — Monitoring key accounts in the IE/SoCal hot chicken market
- **💡 Content Concepts** — AI-curated video ideas based on what's performing
- **📅 Content Calendar** — Weekly post planning + shoot day shot list

## Deploy to Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repo → `app.py` → Deploy
5. Share the link

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
